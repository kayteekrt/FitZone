# 🏋️‍♂️ FitZone

**FitZone** is a beginner-friendly fitness website designed to guide users through different levels of home workouts — Beginner, Intermediate, and Advanced — using simple visual cards and scrollable guides.

---

## 📋 Features

- 🔰 Level-based workout pages (Beginner, Intermediate, Advanced)
- 🖼 Visual exercise cards with images and recommended sets/reps
- 🔄 Smooth scroll navigation to exercise guides
- 📱 Responsive design for desktop and mobile
- 🔐 Login page (static form)
- ℹ️ About page with mission and background
